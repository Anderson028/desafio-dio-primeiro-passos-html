Comunicação entre client e server

voltar

A comunicação entre o cliente e o servidor é quando o cliente acessa uma página pela internet, e o servidor recebe uma requisição e envia para o cliente todo aquele conteúdo, como imagens, aúdios, textos, html, css, e javascript, existe casos que nem necesita da internet sendo assim uma conexão LAN(Local Area Network) solicitando diretamente com o servidor.

O que é HTML?

voltar

HTML(Linguagem de Marcação de HiperTexto) é usado para construção básica de páginas web no qual utiliza-se marcadores(tags)para criar na WEB.

Estrutura HTML

voltar

A estrutura HTML é formado por uma tag que abre todo documento html entre chaves <>, após isso dentro desta tag temos o head que é a cabeça da página porém seus comandos não aparecem no corpo da página, fora da tag head e dentro da tag html vem a tag body que está é o conteúdo que aparece no corpo da página.

O que são tags?

voltar

Tags são os marcadores usados no HTML elas servem para basicamente criar o site. Exemplos tag del Este texto está riscado, mark para marcar uma parte do texto Isto está marcado, sub server para fazer um subtexto exemplo de subtexto

É interessante usar o site w3schools.com para ver mais tags.

O que são atributos?

Voltar

Atributos são comandos que são colocados nas tags para fazer eles terem mais estilos, ids, ou classes. IDs e Classes são um exemplo de atributos que podem ser utilizados na maioria das tagas HTML.
